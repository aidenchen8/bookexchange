#Book Exchange

## Run server

make sure local mongodb run on default port.
run the code in terminal:
`cd server`
`npm i`
`node index.js`

## Run client

run the code in terminal:
`cd client`
`npm i`
`npm start`
Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
